Password Generator


This form allows you to turn your bad, insecure passwords into a good password. The algorithm merely requires the bad password, the account that the password is to be used for, and a "Secret Code" that only you know. The secret code can be the same for all of your passwords. When done, you get a really secure password that becomes your actual password. Simply remember the bad password and the universal "Secret Code," and you will have many secure passwords.

How does it work? Password Generator relies on two algorithms: SHA-256 and Hexacrypt. SHA-256 is what's called a hashing algorithm. It takes an input of any length, and returns a string of random characters. Merely changing one character changes the whole hash in SHA-256. Hexacrypt, the second algorithm, takes the output from SHA-256 and converts it to a string of text characters, instead of simply hex bytes. To read more about Hexacrypt, click Here. The algorithm uses Javascript and jQuery.
